# Mini Duolingo – Vite

## Rodar rápido (recomendado)
1. Instale PNPM (opcional, mas rápido): `npm i -g pnpm`
2. Dentro da pasta, rode:
   ```bash
   pnpm install
   pnpm dev
   ```
3. Abra o endereço mostrado no terminal.

> Dica: Tailwind é carregado via CDN no `index.html` (sem pacotes extras).

## Build / Preview
```bash
pnpm build
pnpm preview
```
